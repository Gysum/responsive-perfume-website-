## 📖 About The Project

**Maison Élixir** is a luxury perfume store homepage designed with a premium dark theme and gold accents. The project demonstrates modern frontend development practices including responsive design, CSS animations, glassmorphism effects, and clean semantic HTML structure.

This project was built as a frontend development task, showcasing skills in:

- Crafting pixel-perfect, visually appealing layouts
- Building fully responsive pages that work across all devices
- Using Bootstrap 5 for layout and components
- Writing clean, maintainable custom CSS
- Implementing smooth animations and micro-interactions

---

## ✨ Features

- **🎨 Luxury Dark Theme** — Elegant dark background with gold accent colors and premium typography
- **📱 Fully Responsive** — Seamlessly adapts to mobile, tablet, and desktop screens
- **✨ Glassmorphism Effects** — Modern frosted glass UI elements with backdrop blur
- **🎬 Scroll Animations** — Elements gracefully reveal on scroll with staggered delays
- **🔢 Counter Animations** — Animated statistics in the hero section
- **🧭 Smooth Scroll Navigation** — Seamless section-to-section scrolling
- **📌 Sticky Navbar** — Fixed navigation with scroll-based style changes and active link highlighting
- **⬆️ Back to Top Button** — Appears on scroll for quick navigation
- **⏳ Preloader** — Branded loading screen for a polished first impression
- **🖼️ Hover Effects** — Interactive product cards and service cards with smooth transitions
- **📝 Working Contact Form** — Form with validation and submit feedback (demo)
- **📧 Newsletter Subscription** — Email input with subscribe confirmation (demo)

## 📄 Page Sections

### 1. 🏠 Hero Section
Full-viewport landing area with a background image, gradient overlay, animated floating particles, brand tagline, call-to-action buttons, and animated statistics (150+ scents, 40k+ clients, 25 years).

### 2. 🏛️ About Section
Split layout with a craftsmanship image and brand story. Highlights the company's heritage, ethical sourcing, and master perfumer expertise with an animated feature list.

### 3. 🛍️ Products / Collections Section
Four product cards showcasing signature fragrances — **Noir Absolu**, **Rose Éternelle**, **Amber Soleil**, and **Crystal Veil**. Each card features hover zoom effects, an overlay quick-view button, pricing, and fragrance descriptions.

### 4. 💼 Services Section
Six service offerings displayed in interactive cards:
- Bespoke Creation
- Luxury Gift Sets
- Scent Discovery
- Global Delivery
- Refill Program
- Private Events

### 5. 💬 Testimonials Section
Client reviews with star ratings, quotes, and author details — building trust and credibility.

### 6. 📬 Newsletter Section
Email subscription CTA with a large background watermark text for visual impact.

### 7. 📞 Contact Section
- **Contact Info Cards** — Address, phone, email, and opening hours
- **Contact Form** — Name, email, phone, subject dropdown, and message textarea with submit feedback

### 8. 🦶 Footer
Multi-column footer with brand info, quick links, collection links, support links, and social media icons (Instagram, Facebook, Twitter/X, Pinterest).

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, Safari)
- No build tools or installation required

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/luxury-perfume-store.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd luxury-perfume-store
   ```

3. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   # Or use a live server extension in VS Code
   ```

> 💡 **Tip:** For the best experience, use the **Live Server** extension in VS Code to avoid any potential CORS issues with local file loading.

## 📱 Responsive Breakpoints

| Breakpoint | Screen Size | Adaptations |
|---|---|---|
| **Desktop** | ≥ 992px | Full multi-column layout, all effects active |
| **Tablet** | 768px – 991px | 2-column grids, adjusted spacing |
| **Mobile** | < 768px | Single column, stacked layout, hamburger menu |
| **Small Mobile** | < 576px | Compact typography, optimized card sizes |
